This GitHub page is a university project on the use of Artificial Intelligence in the field of architecture made by two students in the fifth semester from Hochschule Mainz - University of Applied Sciences.

The hosted page serves as landing page for a web app as per the instructions of the project task, exploring the use of AI, potentials and problems. The app itself aims to identify any problems of a floor plan in concerns to fire safety, adherence to norms (DIN) and inclusion.
